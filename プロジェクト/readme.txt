◆このゲームについて
C++ 3作品目
テンプルランを参考にしたランアクションゲームです。

◆起動方法
exeファイルを起動

◆基本操作
ゲーム
	左右移動		: AD
	ジャンプ		: W
	スライディング	: S
タイトル
	移動	: WS
	決定	: enter

◆実装内容
ストラテジーパターンによる管理シーン
オブジェクトのリスト構造
キャラクターを状態パターンで制御
キャラクターのモーション
オブジェクトのカメラからの距離によるソート
